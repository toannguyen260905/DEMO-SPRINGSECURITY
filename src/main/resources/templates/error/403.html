Từ chối truy cập

🛡️ Spring Security đã chặn request này.
Bạn không có quyền ROLE_ADMIN để truy cập trang này.

Cơ chế hoạt động:

SecurityConfig: .requestMatchers("/admin/**").hasRole("ADMIN")

→ Spring Security phát hiện bạn không có ROLE_ADMIN

→ ExceptionTranslationFilter bắt AccessDeniedException

→ Trả về HTTP 403 Forbidden
